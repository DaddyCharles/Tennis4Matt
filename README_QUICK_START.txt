============================================================
  IVAN - QUICK START GUIDE
============================================================

Welcome! This guide gets you up and running in 3 simple
steps. No coding or technical knowledge needed.


------------------------------------------------------------
  WINDOWS
------------------------------------------------------------

STEP 1 - SET UP (only needed once)
  - Double-click "setup.bat".
  - A black window will open and get everything ready.
    This can take a few minutes - please be patient.
  - If it says Python is missing, the download page will
    open. Install Python (tick "Add Python to PATH"),
    then double-click "setup.bat" again.
  - Wait until you see "Setup complete!".

STEP 2 - START THE APP
  - Double-click the "Ivan" shortcut on your
    Desktop (created during setup).
  - You can also double-click "run.bat" in this folder.

STEP 3 - USE IT
  - Your web browser opens automatically to the dashboard.
  - If it doesn't open, go to:  http://127.0.0.1:9999
  - Keep the black window open while you use the app.
  - To stop the app, just close the black window.


------------------------------------------------------------
  MAC
------------------------------------------------------------

STEP 1 - SET UP (only needed once)
  - Double-click "setup.sh".
    (If it opens in a text editor instead of running,
     right-click it, choose Open With > Terminal.)
  - A Terminal window will open and get everything ready.
    This can take a few minutes - please be patient.
  - Wait until you see "Setup complete!".

STEP 2 - START THE APP
  - Double-click "run.command".
    (The first time, you may need to right-click it and
     choose "Open" to allow it to run.)

STEP 3 - USE IT
  - Your web browser opens automatically to the dashboard.
  - If it doesn't open, go to:  http://127.0.0.1:9999
  - Keep the Terminal window open while you use the app.
  - To stop the app, close the Terminal window.


------------------------------------------------------------
  FIRST TIME IN THE DASHBOARD
------------------------------------------------------------

  - A short setup wizard walks you through connecting your
    Facebook account, choosing groups, and picking keywords.
  - Once setup is done, click "Start Bot" on the dashboard
    to begin monitoring. New leads appear under "Leads".


------------------------------------------------------------
  START AUTOMATICALLY (WINDOWS, OPTIONAL)
------------------------------------------------------------

  - In the app, go to Settings.
  - Under "Start with Windows", click
    "Start automatically when Windows starts".
  - To turn this off later, click "Remove from startup".


------------------------------------------------------------
  NEED HELP?
------------------------------------------------------------

  If anything goes wrong, contact us at:

      support@yourapp.com

  any message you see - it helps us help you faster.
